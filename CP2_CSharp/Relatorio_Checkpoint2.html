2. Descrição da Solução

2.1 Proposta Desenvolvida

Sistema de Gestão de Estoque em camadas utilizando Windows Forms, ADO.NET e SQL Server. A aplicação permite o cadastro, listagem, busca, entrada e saída de produtos, com validações completas e tratamento de erros.

2.2 Arquitetura Adoptada

A solução foi estruturada em 6 camadas bem definidas:

  • UI (Presentation) - Interface Windows Forms
  • BLL (Business) - Regras de negócio e validações
  • DAL (Data Access) - Comunicação com banco via ADO.NET
  • Model - Entidades e objetos de domínio
  • Contracts - Interfaces para inversão de controle
  • Config - Configurações centralizadas

3. Funcionalidades Implementadas

3.1 Funcionalidades Obrigatórias

  • Cadastro de produtos - Nome, preço e quantidade inicial
  • Listagem de produtos - Grid com todos os produtos ativos
  • Busca por nome - Filtro em tempo real
  • Entrada de estoque - Aumentar quantidade disponível
  • Saída de estoque - Diminuir quantidade disponível
  • Impedimento de estoque negativo - Validação implementada
  • Atualização de produtos - Editar dados existentes
  • Exclusão lógica - Soft delete (campo Ativo)

3.2 Validações Implementadas

  • Campos obrigatórios (nome)
  • Preço maior que zero
  • Quantidade não negativa
  • Validação de formato numérico
  • Impedimento de saída maior que estoque disponível
